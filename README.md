# sales-Performance-PowerBI-Dasboard
Interactive Power BI dashboard designed to analyze sales performance, identify key revenue drivers, and track trends across regions, products, and time periods.

## Objective
The objective of this project is to analyze sales data and provide 
actionable insights related to revenue, region-wise performance, 
product contribution, and monthly sales trends using Power BI.

## Tools & Technologies
- Microsoft Power BI
- CSV 
- DAX
  
## Dashboard Features
- Total sales and revenue KPIs
- Region-wise sales analysis
- Product-wise revenue comparison
- Monthly sales trend visualization
- Interactive filters and slicers

## Key Insights
- The highest sales were observed in the North region.
- Product A contributed the maximum revenue.
- Sales showed a noticeable increase during festive months.
